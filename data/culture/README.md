# Culture Database Schema

这个文件夹存放“传统文化解释层”的数据库。

## lodges.json

二十八宿层数据库。键是宿名，例如 `"角"`、`"心"`、`"参"`。

字段：

```json
{
  "name": "角宿",
  "group": "东方苍龙",
  "cycle": "寿星",
  "fenye": "郑之分野，兖州",
  "imagery": "传统意向说明",
  "refs": [
    {
      "explain": "解释这条引用如何支持本条信息",
      "source": "《晋书·天文志》",
      "quote": "古籍原文"
    }
  ]
}
```

## yuans.json

三垣层数据库。键是 `"紫微垣"`、`"太微垣"`、`"天市垣"`。

## asterisms_culture.json

重点星官层数据库。键是星官名，例如 `"北斗"`、`"织女"`、`"河鼓"`、`"天津"`。

推荐字段：

```json
{
  "yuan": "紫微垣",
  "lodge": "牛",
  "region": "星域分野说明",
  "imagery": "传统意向说明",
  "refs": [
    {
      "explain": "解释",
      "source": "古籍名",
      "quote": "原文"
    }
  ]
}
```

如果一个星官没有自己的专属条目，程序会自动回退到所属星宿或三垣。

## stars_culture.json

重点恒星层数据库。键是中文星名，例如 `"织女一"`、`"心宿二"`、`"天枢"`。

如果一个恒星没有自己的专属条目，程序会自动回退到所属星官；如果星官也没有专属条目，再回退到星宿或三垣。
